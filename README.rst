Quickstart
==========

pywavelet is available on PyPI and can be installed with `pip <https://pip.pypa.io>`_.

.. code-block:: console

    $ pip install pywavelet

For developers

.. code-block::

    $ pip install -e .[dev]
    $ pre-commit install
